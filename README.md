#  Student Performance Predictor (AI/ML Project)

## Overview
This project predicts whether a student will pass or fail using Machine Learning based on study habits like study hours, attendance, and sleep.

---

##  Project Type
This project is implemented using **Google Colab (Jupyter Notebook environment)**.

---

##  Technologies Used
- Python
- Pandas
- Scikit-learn
- Google Colab

---

## How to Run 

### Option 1: Run on Google Colab (Recommended)

1. Open Google Colab  
2. Click **File → Upload Notebook**  
3. Upload the `.ipynb` file  
4. Click **Runtime → Run all**

---

### Option 2: Run via GitHub

1. Open the uploaded `.ipynb` file in GitHub  
2. Click **Open in Colab** (if available)  
3. Run all cells  

---

##  Input Features
- Study Hours  
- Attendance  
- Sleep Hours  

---

 Output
- 0 = Fail  
- 1 = Pass  

---

##  Machine Learning Concept Used
- Supervised Learning  
- Classification  
- Logistic Regression  

---
